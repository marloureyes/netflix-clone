--------------------------
Getting Started
--------------------------

1. Clone the repo:
   git clone https://github.com/marloureyes/netflix-clone.git

2. Move inside:
   cd netflix-clone

3. Install the dependencies:
   npm install

4. Create a `.env.local` file with your API key:
   NEXT_PUBLIC_TMDB_API_KEY=your_api_key_here

5. Start the app:
   npm run dev

6. Visit it at:
   http://localhost:3000
